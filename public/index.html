Requirements

To import a course package, it needs to be zipped up (.zip) and needs to be either a SCORM, AICC, xAPI or cmi5 course package.

  • A SCORM package needs to have an imsmanifest.xml file which describes the course and its content.
  • A xAPI package needs to have a tincan.xml file and needs to send xAPI statements to a Learning Record Store.
  • An AICC package needs to be zipped up with the 4 AICC descriptor files (AU, CRS, CST, & DES).
  • A cmi5 package must include a cmi5.xml file which includes a list of assignable units (AU).
